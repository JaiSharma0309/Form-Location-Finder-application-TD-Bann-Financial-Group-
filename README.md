- Automated Retrieval System for Banking Request PDFs

This project details how I developed a side project for a TD Bank branch in Toronto to address an inefficient manual file retrieval process used by the IT department.

The bank maintained a centralized directory containing hundreds of autogenerated subfolders, each with non-meaningful system-generated names (e.g., cbaw.resul.csca.docs0a50cfce589a4fa0a27048a4018847e8). Every subfolder contained the same set of files, including an output.pdf file that stored customer job request information.

When employees required access to a completed job request, they would contact IT and provide only the unique form ID number associated with that request. However, because the directory structure provided no human-readable naming convention, IT staff were forced to manually search through hundreds of subfolders to locate the correct PDF — an extremely time-consuming and error-prone process.

In addition to the PDF file, each folder contained a summary file named FORMS_CTRLFUNC_RPT.DAT, a structured text file that included metadata about the request (form ID, creation date, page count, etc.). This summary file contained the key identifier needed to programmatically locate the correct folder.

- Solution

I designed and implemented an automated retrieval system using a combination of Unix shell scripting and Java.

The system works as follows:

The IT user runs a script with two arguments:

The path to the central directory

The unique form ID number

The script iterates through all subfolders and passes each FORMS_CTRLFUNC_RPT.DAT file to a Java program.

The Java program parses each file using BufferedReader and StringTokenizer to efficiently extract and compare form IDs.

Once a match is found, the program returns the corresponding folder path.

Since the required output.pdf resides in the same folder, the IT staff can immediately retrieve and send the file.

- Impact

This solution eliminated the need for manual folder-by-folder searching, significantly reducing retrieval time and improving operational efficiency.
